# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a respectful, inclusive, and harassment-free experience for everyone.

We are committed to fostering an open environment that encourages constructive discussion, collaboration, and mutual respect.

---

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and considerate in language and actions
- Providing constructive feedback in good faith
- Focusing on what is best for the community and the project
- Showing empathy toward other community members

Examples of unacceptable behavior include:

- Harassment, discrimination, or exclusionary behavior
- Trolling, insulting, or derogatory comments
- Personal attacks or sustained disruption of discussions
- Publishing othersâ€™ private information without permission

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues, pull requests, and discussions
- Project documentation and code reviews
- Any other forums or channels officially associated with this project

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Unacceptable behavior may result in actions such as:

- Requesting corrective behavior
- Removing comments, commits, or contributions
- Temporarily or permanently restricting participation

---

## Reporting

If you experience or witness unacceptable behavior, please report it privately via GitHub by contacting the project maintainers.

Reports will be reviewed and handled in a respectful and confidential manner.
